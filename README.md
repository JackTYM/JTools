# JTools

A simple text tool program made in Python

## Installation

Install requirements

```bash
pip install -r requirements.txt
```

## Usage

Open the main program (JTools.py)

OR

Open an individual program

List Creator:
```  
  placeholder: x
  format: print('x')
  output mode: 1
  input mode: 1
  list: abcd
  
  output: "
  print('a')
  print('b')
  print('c')
  print('d')"
```

